# AILEAP デザインガイドライン

## 1. ブランドコンセプト

### 1.1 ブランドパーソナリティ

| 属性 | 説明 |
|------|------|
| **先進的** | 最新のAI技術を体現 |
| **信頼感** | プロフェッショナルで安心感がある |
| **躍動感** | 「LEAP（飛躍）」を表現 |
| **親しみやすさ** | 中小企業・個人事業主に寄り添う |

### 1.2 ブランドメッセージ

- **タグライン**: 「始めるなら、今。」
- **サブライン**: 「AIを味方に、ビジネスを次のステージへ」
- **ブランドプロミス**: 自立してAIを活用できる力を身につける

## 6. アニメーション

### 6.1 イージング

```css
/* 標準 */
--ease-default: cubic-bezier(0.4, 0, 0.2, 1);

/* スムーズ */
--ease-smooth: cubic-bezier(0.65, 0, 0.35, 1);

/* バウンス */
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);

/* 急速に減速 */
--ease-out-expo: cubic-bezier(0.16, 1, 0.3, 1);
```

### 6.2 デュレーション

```css
--duration-fast: 150ms;    /* ホバー、トグル */
--duration-normal: 300ms;  /* 標準トランジション */
--duration-slow: 500ms;    /* モーダル、パネル */
--duration-slower: 800ms;  /* ページ遷移 */
```

### 6.3 アニメーションパターン

```
【フェードイン + 上昇】
- opacity: 0 → 1
- translateY: 60px → 0
- duration: 800ms
- ease: power3.out

【スケールイン】
- scale: 0.9 → 1
- opacity: 0 → 1
- duration: 500ms
- ease: back.out(1.7)

【テキストスプリット】
- 各文字を個別にアニメーション
- stagger: 20ms
- rotateX: -90deg → 0
- y: 100px → 0

【スクロール連動】
- ScrollTrigger使用
- start: "top 80%"
- scrub: false（トリガー式）
```

### 7.2 レスポンシブ対応

```
【Hero】
- Mobile: 中央寄せ、フォントサイズ縮小
- Desktop: 左寄せまたは中央

【Grid】
- Mobile: 1カラム
- Tablet: 2カラム
- Desktop: 3カラム

【Navigation】
- Mobile: ハンバーガーメニュー
- Desktop: 横並びメニュー

【Three.js】
- Mobile: ノード数50%削減、dpr=1
- Desktop: フル表示、dpr=2
```
