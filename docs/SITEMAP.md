## URL設計

| ページ | URL | 備考 |
|--------|-----|------|
| トップ | `/` | シングルページ |
| トップ（セクション） | `/#about`, `/#services`, etc. | アンカーリンク |
| Works一覧 | `/works` | 全プロジェクト表示 |
| Works詳細 | `/works/[slug]` | 例: `/works/psy-logic-core` |
| プライバシーポリシー | `/privacy` | 必要に応じて |
| 404 | `/404` | カスタム404ページ |
